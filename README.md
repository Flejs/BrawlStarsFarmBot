# BrawlStars Farm BOT


## Description
It's a simple farmbot that plays brawl stars duo showdown

## Getting Started

### Dependencies
* tkinter
* threading
* pyautogui
* Windows 10


### Executing program

* Open brawlstars on bluestack
* Center screen and run
* Must be in window mode by def settings
